# Raw data

This folder is intentionally excluded from version control.

The analysis uses publicly available datasets, including:
- GWAS summary statistics
- GTEx liver eQTL data
- TCGA-LIHC bulk RNA-seq data

Please download the relevant source files and place them here before running the scripts.

Expected filenames:
- `GCST90809296.tsv`
- `GTEx_liver_eqtl.csv`

Controlled-access tokens or private credentials should never be committed.
